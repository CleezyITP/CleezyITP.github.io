Screen Reader Web Development

Welcome to our Web Development Course and Study for screen reader!

This course will be held between 1pm and 4:30pm, weekdays between August 15th and 24th at the Andrew Heiskell Library (40 West 20th St, New York, NY 10011). Students will be asked to complete a survey to help us improve the course.

Schedule Overview

  1. Wednesday: Intro to the web (websites, browsers and the net), web languages, file structure, text editor and other software to be used.
  2. Thursday/Friday: HTML and build a basic site
  3. Monday: HTML refresher and Intro to CSS
  4. Tuesday: CSS
  5. Wednesday: CSS refresher and Intro to JavaScript
  6. Thursday: JavaScript and uploading a site to a server
  7. Friday: Work on sites
