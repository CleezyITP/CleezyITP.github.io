CSS stands for Cascading Style Sheet, and CSS describes the visual style of a page. If HTML is the skeleton of your webpage or defining structure, CSS is the skin or visual appearance. The syntax of the language is different from HTML and there are a number of different ways to use it.

Syntax

When incorporating CSS in your website, there is a specific way the code is formatted. You must write

  1. the element type/selector
  2. a left curly brace,
  3. whatever visual aspect you want to change (for example, the font family)
  4. a colon
  5. a value
  6. a semi-colon
  7. right curly brace
For example, changing the entire background of my site to the color red will be written as the following: body { background-color : red ; }

How to incorporate CSS into your HTML

Here are the three ways that you can include CSS and visual styles into your websites:

  1. You can link to an external CSS file and link to the external file from (External style sheet)
  2. You can write CSS in the head of your page (Internal style sheet)
  3. You can write CSS as an attribute of an element (Inline style)

Ways of using CSS: Selectors

CSS can be used to style a page in four different ways.

  1. Element selector: You can apply a style to every element of a paricular type. For example, you can have the text in every paragraph tag be colored purple:
    2.                 p{
                color:purple;
            }
  2. Class: You can apply visual style to a specific html elements by adding the class attribute. You may call your class by any name, as an example, you can write class attributes like this: class="moving". Classes allow you to target specific elements and modify them. This is often used with the div element for layout purposes.
  3. ID : You can also apply visual style to a specific html elements by adding the I D attribute. I Ds give you the ability to change style and content using Javascript. I Ds are useful for adressing specific elements with unique I D s and therefore, you should only use an I D once.
  4. Pseudo class : you can apply a style to elements when an action happens. For example, when you select a link, you can italicize the text.

Element selector

in HTML:

<h1>Home Page</h1>

in CSS:

h1 { font-style : italic ; color : blue ; }

Class selector

in HTML:

<p class=information>...</p>

in CSS:

.information { font-face : san-serif ; background-color : aliceblue ; }

ID selector

in HTML:

<p id="update">....</p>

in CSS:

#update { border-style : dashed ; border-width : thin ; }
