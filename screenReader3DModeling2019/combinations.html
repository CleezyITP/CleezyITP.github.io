Transformations

Union

Union joins together objects and is the sum of all shapes between its brackets.  

union() {
  cylinder(h = 40, r = 10, center = true);
  rotate([90, 0, 0])
    cylinder (h = 40, r = 9, center = true);
}

Difference

Difference is using the second shape (and all subsequent shapes in the bracket) to cut out from the first shape. 

difference() {
  cylinder(h = 40, r = 10, center = true);
  rotate([90, 0, 0])
    cylinder(40, r = 9, center = true);
}

Translate (Move)

Translate helps you move objects in the workplane. Translations are written as follows: translate( [x-value, y-value, z-value]);

Positive X values, move objects to the right. Negative X values, move objects to the left. Positive Y values, move objects away from the user. Negative Y values, move objects toward the user. Positive Z values, move objects up. Negative Z values, move objects down.

  1. This is how you move an object away from the center of the workplane: 
    ￼
translate( [0, 0, 20])
      sphere(r = 20);
  2. The primitive below a translate line and before the semi-colon that is found at the end of the sphere, gets moved. 
    ￼
translate( [-20, 0, 20])
  cube( [30,30,30]);
cube([2,3,4]); translate([3,0,0]) { cube([2,3,4]); }

Rotate

Rotates its child 'a' degrees about the axis of the coordinate system or around an arbitrary axis. The argument names are optional if the arguments are given in the same order as specified.

rotate(a = deg_a, v = [x, y, z]) { ... }
  1. The following will an object 45 degrees along the x axis: 
    rotate([45, 0, 0])
  cube([10, 20, 15]);

Parameters

  • "x" is a rotation about the X axis, from the +Y axis, toward the +Z axis.
  • "y" is a rotation about the Y axis, from the +Z axis, toward the +X axis.
  • "z" is a rotation about the Z axis, from the +X axis, toward the +Y axis.

Point your right thumb along the positive axis, your fingers rotate in the direction of the rotation.

Scaling

Scaling will help you bend and shape your primitives. The following example creates a vertically elongated egg shape. 

scale([1, 1, 1.5])
sphere(r = 10);

Mirror

mirror(v= [x, y, z] ) { ... }


mirror([1,0,0])cube([2,3,4]);
