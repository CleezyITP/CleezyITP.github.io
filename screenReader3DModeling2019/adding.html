Transformation Review

Rotate

Rotates its child a number of degrees about the axis of the coordinate system.

rotate ([90,0,0]) //rotate a cylinder 90 degrees on the x-axis rotate ([90,0,0]) cylinder (h = 4, r=0.9, center = true, $fn=100);

Righhand Rotation Rule

  • "x" is a rotation about the X axis, from the +Y axis, toward the +Z axis.
  • "y" is a rotation about the Y axis, from the +Z axis, toward the +X axis.
  • "z" is a rotation about the Z axis, from the +X axis, toward the +Y axis.

Point your right thumb along the positive axis, your fingers rotate in the direction of the rotation.

Union

Creates a union of all its child nodes. This is the sum of all children (logical or). May be used with either 2D or 3D objects, but don't mix them.

union() { cylinder (h = 4, r=1, center = true, $fn=100); rotate ([90,0,0]) cylinder (h = 4, r=0.9, center = true, $fn=100); }

Difference

difference() { cylinder (h = 4, r=1, center = true, $fn=100); rotate ([90,0,0]) cylinder (h = 4, r=0.9, center = true, $fn=100); }

Translate (Move)

Translates moves its child elements along the specified vector. The argument name is optional. translate(v = [x, y, z]) { ... } cube([2,3,4]); translate([3,0,0]) { cube([2,3,4]); }

Mirror

mirror(v= [x, y, z] ) { ... }


mirror([1,0,0])cube([2,3,4]);
